Primitive Print Generator

The purpose here is to generate random coloured bars on a 'canvas'. Clicking the START button will run the process and generate a 'print'. It is possible to run the sequence any number of times and get completely different finished prints or 'paintings'. Clicking the SAVE button will save the page as a png file, and the CLEAR button will revert to a blank canvas.

My intention is to create a number of painted images, and randomness is a key part of the process. I have created this app to help with this aspect. A by-product is the generation of 'prints' which exist as artworks in their own right. For those who wish to actually paint one of the resulting images the colour code and depth for each of the bars are listed below the buttons.
